## Quick Start
1. Export OBD-II data as CSV/TXT with fuel trim values
2. Load file into the tool
3. Analyze results for potential vacuum leaks

## Features
- Automatic CSV parsing and column detection
- Configurable RPM thresholds and sensitivity
- Real-time analysis with visual results
- Client-side processing (no data uploaded)
